Detailed Description

Services for random target and default positions.

Author
Francesco Ganci (S4143910)
Version
1.0
Date
2021-06-25

This simple ROS node provides 2 services:

  • /get_point : it returns a randomly choosen target amond the predefined targets.
  • /position_defined : it returns true if the sent point belongs to the set of predefined ones.
